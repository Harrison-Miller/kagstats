About

King Arthur's Gold

King Arthur's Gold (KAG) is Free to Play 2d, action, platforming, medieval themed multiplayer game. You can learn more about the game on the kag2d.com website.

KAG Stats

KAG Stats is a website providing real time updates of basic stats from KAG Servers. KAG Stats connects to servers using the rcon login functionality and watches for messages from a script that outputs information about kills and important events. On the website you can find many leaderboards, breakdowns of a player's stats, a live kill feed and much more.

Anybody can host their own KAG stats websites for their own servers! The main website for official KAG servers is at kagstats.com.

Development

KAG Stats is an opensoure piece of software that anybody is free to use and modify for their own purpose. The repository is hosted on github. Anybody may fork the repository and submit PRs to improve the quality of the software. If you want to contribute but don't know where to start, see the trello board listing tasks.
